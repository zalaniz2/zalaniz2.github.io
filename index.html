Jazz MIDI Application (JMA)

Node.js, MongoDB, Javascript, Web MIDI Api

jma repo

JMA is an application built in Node.js that connects to MIDI controllers (keyboards, etc.) and runs users through excercises based on jazz theory. Each exercise is timed and provides feedback which includes wrong notes, correct/incorrect inputs, as well as accuracy and how long it took. Each exercise is stored and called upon from a Mongo Database, and which notes are being played are recieved through the Web MIDI Api (reference). Feel free to check out the repo for more information.

Floor Bot (Discord Bot)

Java, MySQL, JDA, JDBC

floor bot repo

Floor Bot is a discord bot created in java that simulates a multiplayer RPG game within a discord server. It supports 25+ commands that range from combat to trading. Player and game information is created and stored in a SQL database. JDA (reference) is a Java wrapper for the discord api. Check out the bot repo for more information on how the game works and commands.
